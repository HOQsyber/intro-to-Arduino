# intro-to-Arduino

صفحة تعليمية تفاعلية لورشة أردوينو قصيرة.

## المحتويات
- `arduino-camp.html`: الصفحة الرئيسية للورشة.
- تحتوي على شرح الأوامر، التمارين، بطاقات المشاريع، ومرجع سريع.

## التشغيل
1. افتح `arduino-camp.html` في المتصفح.
2. تصفح الأقسام من البداية إلى النهاية.
3. استخدم الصفحة كمرجع تعليمي للأوامر والمشاريع.
